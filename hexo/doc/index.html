hexo+butterfly 搭建个人博客

hexo的安装

官网

​ 网址：https://hexo.io/zh-cn/docs/

安装git

1. 一个代码版本控制工具
 2. 下载主题theme时使用
 3. 发布自己的个人博客到github时使用

安装node.js

node.js是hexo的运行环境，我本地之前安装过了，所以就直接更新它。

node.js 版本限制：

​ https://hexo.io/zh-cn/docs/#Node-js-%E7%89%88%E6%9C%AC%E9%99%90%E5%88%B6

windows下更新node.js: 

 1. 在GitHub官网，搜索gnvm，具体地址：<https://github.com/Kenshin/gnvm>，进入下载【gnvm.exe】可执行文件
 2. 将这个可执行文件放到自己安装的nodejs的根目录下，如 C:\Program Files\nodejs
 3. 打开cmd命令行窗口，输入：`gnvm update latest`，等待更新即可
 4. 更新成功后，可以该目录下新增了一个以版本号命名的文件夹，将该文件夹下面的node.exe取出，替换之前版本的node.exe
 5. 在cmd命令行窗口，输入 `node -v`  查询node的当前版本

安装hexo

  1. 下载hexo : npm install -g hexo-cli
  2. 添加环境变量： echo 'PATH="$PATH:./node_modules/.bin"' >> ~/.profile

使用butterfly主题

官网：https://butterfly.js.org/

官网描述的很详细，可以直接去官网看操作步骤

创建站点

  1. hexo init site(站名称)
  2. cd site
  3. npm install

下载主题

​ 到themes目录下，使用git clone下载主题

git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git

应用主题

​ 在 Hexo 根目录下，你新创建的站点下，修改_config.yml文件

theme: hexo-theme-butterfly

安装插件：

​ 如果你没有 pug 以及 stylus 的渲染器，请下载安装：

npm install hexo-renderer-pug hexo-renderer-stylus –save

启动服务器：

hexo server

​ 启动服务器。默认情况下，访问网址为： http://localhost:4000/。

发布到github上

修改发布配置：

​ 文件：主题目录下的_config.yml文件
